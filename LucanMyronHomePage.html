About the Job:

Job Description

Software Developer is sort of a blanket term. The duties of a developer can range from creating software and applications, designing machine code, developing video games, updating existing software, and so much more. A good software developer is knowledgeable in a few different coding languages. While the don't typically need to be an expert it is preferred that developers, new and seasoned, be familiar with concepts like software development life cycles (SDLCs), Object Oriented Programming, testing, and more.
